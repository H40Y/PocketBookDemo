路遇坎坷

Access denied for user 'root'@'localhost' (using password: YES)

1、检查用户名密码是否正确
2、检查对应用户的访问权限（localhost、%）
3、修改my.ini，启动服务
4、服务名不一定是mysql，要参照计算机管理里面的服务名

Before start of result set

即使确保JDBC只查询到一条语句，也要使用.next()来读取第一条

onsubmit

表单提交的时候是会触发onsubmit事件的，如果onsubmit事件的方法返回是true，那么该表单允许提交，如果返回的是false，该表单不允许提交。
此外，网上复制的内容不一定完全正确（尤其是英文，就算看起来是对的），可能夹带着不同的代码导致运行出错，最好自己手动输入code。

<!DOCTYPE html>

HTML5的声明。
除了这个以外的诸如<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">是HTML4及以下版本的声明，用错可能会报错，如“Undefined attribute name (placeholder) ”。

NullPointerException

使用JAVA最容易出现的东西，原因可能在于1、未实例化2、值为空就去比较（如equal操作或者==）3、越界
第二天再度出现了这个问题，不过原因是表单数据没有接收到。一个表单控件可以定义很多属性，type、id、name、class，getParameter获取的是name，name!!!

method <init>()V not found

javabean在没有数据的时候调用的是无参的构造函数，所以一旦定义了有参的构造函数，需要补一个无参的构造函数。
